# Datasets — MEDIETHIC / NSFWGuard / MEDIETHIC-600

Three datasets accompany the paper *"A Safety-aware Jailbreak Approach to Bypass Security Filters
for Generating Harmful Media with Text-to-Image Models"*. All are released **gated** (request access)
under **CC BY 4.0**, for research on T2I safety only.

> ⚠️ These datasets contain harmful, sexual, violent and otherwise sensitive synthetic imagery and
> text, generated solely for safety-evaluation research. Access is restricted to verified academic /
> industrial researchers. Do not redistribute the images.

## Overview

| Dataset | Role | Size | Modality |
|---|---|---|---|
| **NSFWGuard** | Training set (safety classifier ψ + harm classifier) | **50,000** prompt–image pairs (balanced safe/unsafe) | prompt + image |
| **MEDIETHIC** | Adversarial images, human-annotated | **5,775** images (DALL·E 3: 4,126 · IMAGEN 3: 1,649) — **6,634** incl. Nanobanana (859) | prompt + image + ratings |
| **MEDIETHIC-600** | Test set (curated unsafe prompts) | **588** prompts (98 base × 6 dilutions) | prompts only |

7 harm categories: harassment, hate, illegal-activity, self-harm, sexual, shocking, violence.

## NSFWGuard (training, 50k)
- **Construction (paper §5)**: 70k LLM-generated prompts (uncensored LLMs, AIR-2024 taxonomy) →
  majority-vote filtering (OpenAI + Azure + Google moderation) → 20k synthetic images (NewRealityXL) →
  + 30k LAION image–prompt pairs → **50,000 balanced pairs**.
- **Composition (released)**: 29,418 LAION (`REAL`) + 20,582 synthetic (`SYNTHETIC`), balanced 25k safe / 25k unsafe.
- **Provenance note**: the original submission-era 50k snapshot was not retained; this release is a
  **deterministic reconstruction** (LAION component is exact; synthetic subset selected by category/label
  balance, seed=42). See `scripts/build_nsfwguard_50k.py` + `provenance.json` + `selected_ids.csv`.
  The image-bearing version is built by filtering the source shards (`scripts/build_nsfwguard_images.py`).
- **Schema (key cols)**: `prompt, label (0/1), category, source, type (REAL/SYNTHETIC), image, GLOBAL_ID_OUR`.

## MEDIETHIC (adversarial images, 5,775 + Nanobanana)
- Images generated by the Safety-Aware PGD attack on **DALL·E 3 (4,126)** and **IMAGEN 3 (1,649)**,
  each rated by **3 annotators** (Likert 1–5; harmful if mean ≥ 3).
- **Nanobanana extension (R2.2)**: **859** images on **Gemini 2.5 Flash Image** (Vertex AI), evaluated by
  the Gemini vision evaluator (`harm_level` 1–5; harmful if > 3). Merged into MEDIETHIC → **6,634** total
  (`t2i_model = "Gemini 2.5 Flash Image"`). The Nanobanana rows carry `human_rating` from the vision
  evaluator and have empty human-annotator columns (no 3-rater human annotation for this subset).
- **Schema**: `category, t2i_model, adv_prompt, base_prompt, unsafe_prompt, safe_prompt, human_rating,
  classifier_score, clip_score, label, annotator_1, annotator_2, annotator_3, image_id, image`.
- Build: `scripts/build_mediethic_paper.py` (metadata) + `scripts/build_nanobanana_images.py` (images).

## MEDIETHIC-600 (test, 588)
- Curated unsafe test prompts: GPT-4 generates 100/category → K-means (Silhouette → 14 clusters) → 98
  base prompts → dilution ×6 → **588** prompts (released version: 585 after cleaning).
- Schema: `category, prompt, unsafe_prompt, safe_prompt, unsafe_words, safe_words, …`.

## Hosting & access
- Image-bearing parquet shards (large) are hosted on a **gated HuggingFace dataset** (request access),
  split into two configs: **`mediethic_nsfw`** (prompts + images) and **`mediethic_adv`** (prompts + images).
  Embeddings are **not** distributed.
- This code repo does **not** ship the heavy data nor the data-preparation scripts; the datasets are
  obtained from the gated repository (access granted to verified researchers).

## De-rebranding note
This dataset is the paper-aligned version of material that was later reworked, under the name **MEDIETHIC**,
for a separate benchmark paper. Here it uses the paper's original naming (MEDIETHIC / NSFWGuard /
MEDIETHIC-600) and is sized to the values reported in this submission.
